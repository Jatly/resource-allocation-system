# 🏢 Resource Allocation System

A full-stack MERN application to efficiently manage and allocate office resources like meeting rooms, workspaces, and equipment with conflict detection and role-based access control.

---

## 🚀 Features

- 🔐 User Authentication (Signup/Login with JWT)
- 👤 Role-Based Access (Admin & User)
- 📅 Resource Booking System
- ⚠️ Conflict Detection (No overlapping bookings)
- 💾 Auto-save & persistent storage
- 📊 Admin Dashboard (Manage users & resources)
- 🔎 View availability in real-time
- 🔗 Shareable booking system (future scope)

---

## 🧠 Problem Solved

In many offices, resource booking is messy and leads to:
- Double bookings ❌
- Poor visibility ❌
- Manual coordination ❌

This system ensures:
- Conflict-free scheduling ✅  
- Centralized management ✅  
- Efficient resource usage ✅
